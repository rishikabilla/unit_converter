# Unit Converter Web App

A simple Flask-based unit converter that supports:
- Length conversion
- Weight conversion
- Temperature conversion

## Features
- Convert between multiple units
- Clean UI with CSS
- Separate pages for each converter
- Reset functionality without losing context

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Project Structure

- main.py
- README.md
- templates/
  - index.html
  - weight.html
  - temp.html
- static/
  - style.css



## How to Run
